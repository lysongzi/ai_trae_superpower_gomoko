# Finishing a Development Branch

Confirm all tests pass, present merge/PR/keep/discard options, and clean up temporary worktrees or branches to maintain a tidy repository state.

## Triggers
- All planned tasks complete with green tests.

## Guidance
- Verify test suite; ensure no regressions.
- Decide merge or PR; document decision.
- Remove temporary worktrees; tidy branches.

## Output
- Clean repository with changes integrated or staged.
