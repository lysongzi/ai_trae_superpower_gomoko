# Writing Plans

Create clear, atomic tasks (2–5 minutes each) that include exact file paths, complete code snippets, and verification steps to enable autonomous execution.

## Triggers
- Design is approved and ready for implementation.

## Guidance
- Each task must have a specific outcome and verification.
- Prefer small steps; avoid speculative work (YAGNI).
- Ensure tasks are independently runnable.

## Output
- A task list that subagents can execute.
